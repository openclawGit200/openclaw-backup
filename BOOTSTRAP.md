# BOOTSTRAP.md

## Session 啟動第一步
讀取 `MEMORY.md`（workspace 根目錄），了解：
- 今日重點摘要
- 正在進行的專案（一定要記得！）
- 已安裝技能
- 重要憑證位置
- 尚未解決的問題

**⚠️ 最低標準：Session 開始時，說出目前在進行中的專案名稱與進度。**

## Session 中
依據觸發關鍵字，讀取對應的 LEARNINGS 或專案 MD：
- 觸發「技能」、「SKILL」→ 讀取 MEMORY.md 的「已安裝技能」區段
- 觸發「錯誤」、「失敗」→ 讀取 `.learnings/ERRORS.md`
- 觸發「學習」、「犯錯」→ 讀取 `.learnings/LEARNINGS.md`
- 觸發特定專案（台股、CF、GitHub）→ 讀取 `notes/projects/{專案}.md`
- 觸發「記憶」、「之前說過」→ 先讀取 MEMORY.md，再依關鍵字查對應檔

## Session 結束前（主動執行，不等提醒）
- 主動問：「要歸檔嗎？」
- 若用戶同意，把本次決策與進度寫入 MEMORY.md 或對應的 notes

## 禁止
在讀完 MEMORY.md 之前，開始執行任何操作
